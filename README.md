  <h2>About the Project</h2>
  <p>
    Crown & Glow Hair Salon is an online salon management system that allows customers
    to register, log in, and book appointments, while providing administrators with
    a dedicated panel to manage customers and appointments efficiently.
  </p>

  <hr>

  <h2>Features</h2>
  <ul>
    <li>Modern and user-friendly salon landing page</li>
    <li>Customer registration and login system</li>
    <li>Session-based authentication using PHP</li>
    <li>Online appointment booking system</li>
    <li>Dedicated admin panel</li>
    <li>Admin management of customers and appointments</li>
    <li>Database-driven backend using MySQL</li>
  </ul>

  <hr>

  <h2>Customer Features</h2>
  <ul>
    <li>Create and manage personal accounts</li>
    <li>Log in securely</li>
    <li>Book salon appointments online</li>
    <li>View booking information</li>
    <li>Access salon contact details and social media links</li>
  </ul>

  <hr>

  <h2>Admin Panel</h2>
  <p>
    The project includes an admin section that enables salon administrators
    to manage daily operations through the system.
  </p>
  <ul>
    <li>View all registered customers</li>
    <li>View all booked appointments</li>
    <li>Monitor and manage appointment schedules</li>
    <li>Access customer and booking data from the database</li>
  </ul>

  <h2>How to Run Locally</h2>
  <ol>
    <li>Install XAMPP, WAMP, or MAMP</li>
    <li>Move the project folder into the <code>htdocs</code> directory</li>
    <li>Create a MySQL database named <code>crown_and_glow_salon</code></li>
    <li>Configure database credentials in <code>API/db_connect.php</code></li>
    <li>Start Apache and MySQL</li>
    <li>Open the project in your browser</li>
  </ol>
